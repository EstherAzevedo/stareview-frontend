# 📚 StaReview – Frontend

StaReview é uma aplicação web voltada para **resenhas literárias**, permitindo que leitores compartilhem opiniões, descubram novos livros e interajam através de avaliações e comentários.

## 🚀 Tecnologias
- **Vue 3**
- **TypeScript**
- **Vite**
- **Pinia**
- **Vue Router**
- **Tailwind CSS**
- **Axios**
